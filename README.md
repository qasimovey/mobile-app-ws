# Restful Web Service with Spring Boot

Spring Starter Web ,project lombok , jackson-dataformat 

## Major changes happened in versions

  1.0.0 :

 
 - Exception Handler is added
 - List is used to store data

## Installation


```bash
git clone [url]
mvn spring-boot:run
```

## Usage

 - GET - Getting resource     -- /api/users/ or /api/users/{userID}
 - PUT - Creating new resource -- /api/users/
 - POST - Updating resource   -- /api/users/{userID}
 - DELETE -Remove resource   --  /api/users/{userID}
